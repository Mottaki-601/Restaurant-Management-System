# 🍔 Restaurant Management System

A modern, scalable **ASP.NET MVC 5 web application** for managing fast food operations including categories, menu items, and stock with dynamic UI interactions and secure authentication.

---

## ✨ Project Summary

This project is designed to streamline fast food business management by providing a centralized platform where administrators can:

- Manage categories and food items
- Handle stock dynamically (size, price, quantity)
- Upload and manage food images
- Perform operations securely with role-based access

Built using **ASP.NET MVC + Entity Framework + SQL Server**, this system demonstrates real-world enterprise development practices.

---

## 🚀 Core Features

### 🔐 Authentication & Security
- ASP.NET Identity integration
- Login / Register / Logout functionality
- Role-based authorization (Admin control)
- Anti-forgery token protection

### 📦 Management Modules
- Category Management (Full CRUD)
- Food Item Management (Full CRUD)
- Fast Food Menu Management

### ⚡ Advanced Functionalities
- Dynamic Stock Entry (Add/Remove rows)
- Stored Procedure integration for performance
- Transaction handling for data consistency
- AJAX-based Delete (no page reload)
- Image Upload with server storage

### 🎨 UI/UX
- Clean and modern dashboard
- Card-based design
- Responsive layout (Bootstrap)
- Interactive UI with jQuery
